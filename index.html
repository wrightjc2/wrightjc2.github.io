SKSE Documentation

Introduction

  • SKSE is a script extender required to many user created modifications in the Bethesda Title “The Elder Scrolls V: Skyrim”
  • This documentation page will walk you through the basic steps of installing SKSE yourself without an installer
  • Will also describe how to test whether or not it has been installed correctly

Installation

Requirements

  • Valid, Legal copy of a PC version of Skyrim (Classic, Special Edition, VR)
  • SKSE - SKSE Download Page

Steps

  1. First, navigate to the SKSE Homepage and download the appropriate 7z folder for your Skyrim version
  2. Unzip the folder
  3. Open your Skyrim game folder. If you have a steam version the filepath for this is usually Steam\steamapps\common\skyrim. This filepath may vary depending on if you manually chose the install location or received Skyrim from a different storefront. The file should contain the Skyrim.exe executable.
  4. Copy each of the .dll extension and .exe files into this directory.
  5. Go into the Data Folder then Go into the Scripts Folder. Copy all .pex extension files into this folder
  6. Launch Skyrim either through steam or through clicking the skse_loader.exe
  7. Press the (~) key to bring up the console. Then type (getskseversion) followed by enter. If the skse version appears, your install was succesful

TroubleShooting

  • SKSE not launching with Skyrim:
    -Run skse_loader.exe as an administrator by right clicking on the executable and clicking “Run as Administrator”.

  • I did not buy my copy of Skyrim from Steam:
    -The steps are fairly similar regardless of where your copy was purchased. The only real difference will be that the filepath to the game will be slightly different. A good way to do this is to inspect the install location on the games page by clicking the gear icon, then selecting properties, and then browse. This should bring you to the games file location.

  • A mod I installed is giving me a message like “SKSE Not installed Correctly” or “Old version of skse found”:
    -The first thing to check is if the version of skse you downloaded is the correct version for the copy of skyrim you have. Again, you must download the skse file that correlates with the version of skyrim you have. If you have checked, and the version is correct, make sure that you have the most up to date version of SKSE offered on the skse website for your version.

FAQ

  • Q: Why is SKSE necessary for so many mods?
    -SKSE provides a plethora of tools for modders including expanded scripting capabilities that essentially let modders pull off amazing user created modifications that would not be possible or would be much more difficult without it.

  • Q: Can I download SKSE straight from my favorite modding website (nexus, etc)?
    -Unfortunately no. Any mod on these sites that claims to be the “Official SKSE” is not the actual script extender. The official version can only be found on the website linked above.
